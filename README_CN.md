# YeapCaptcha
Yeap Cloud security captcha.

# APP ID/Secrect Key
登录Yeap Cloud控制面板，进入“安全”>“验证码”中，创建APP或者找到已经创建的APP的ID和密钥。

# 语言
目前仅支持中文和英文，后续会支持更多语言，也欢迎主动提交翻译开发者们。

# 验证码类型
现在仅有三种：rotate（旋转，默认类型），click（点选文字），dx（图块）

后两者目前只支持中文，后续根据需求开发其他语言类型支持
